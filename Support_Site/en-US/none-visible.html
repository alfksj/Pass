Why no one is shown on sharable list?

Learn what to do if no one found.

There are many reasons that makes this problem. Check below and let's solve it!

1. Check there're sharable computer in your network

First of all, check if there're sharable computer in your network. "sharable computer" means the computer that satisfy these conditions:

  • Running Pass program
  • Both Pass program is same version
  • User allowed sharing

If problem keep occur even there're sharable computer, go to 2.

2. Check internet setting

Check all below things and check if your computer is well-set.

  • Is both computer connected to same router(same private network)?
  • Is firewall allow Pass to perform UDP protocol in your current network profile(Public/Private)?
  • Is your computer connected to WLAN or LAN properly?

Still have problem?

In this situation, you would be better see log. You can see log as following way.

  1. Open Pass and focus on it
  2. Press F9 5 times in a row
  3. Refresh sharable computer list
  4. After refresh is done. IP address and response will be shown.

When you see "Response", will find what is the problem and why certain computer denied your request.
